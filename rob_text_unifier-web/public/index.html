🔥 Funcionalidades Poderosas

Tudo que você precisa para preparar seu código para IAs

📁

Filtro por Nome

Ignore arquivos específicos pelo nome exato como package-lock.json, .gitignore, etc.

🗂️

Filtro por Pasta

Pule pastas completas como node_modules, .git, __pycache__ e muito mais.

📄

Filtro por Extensão

Ignore todas as arquivos de uma extensão: .log, .tmp, .bak, .pyc.

🎯

Regra Parcial

Ignore arquivos soltos em uma pasta, mas continue lendo suas subpastas. Controle total!

📊

Caminhos Relativos

Mantém a estrutura de pastas no arquivo de saída para fácil navegação e entendimento.

🛡️

Tratamento de Erros

Continua executando mesmo se algum arquivo não puder ser lido. Sem travamentos!

⚡ Como Funciona

3 passos simples para ter seu código consolidado

1

Baixe o Script

Clone o repositório ou baixe o arquivo rob_text_unifier.py

git clone https://github.com/SEU_USUARIO/rob_text_unifier.git
2

Configure os Filtros

Edite as listas de arquivos, pastas e extensões a ignorar

arquivos_ignorados = {"package-lock.json"}
pastas_ignoradas = {"node_modules", ".git"}
extensoes_ignoradas = {"log", "tmp"}
3

Execute e Use

Rode o script e copie o resultado para sua IA favorita

python rob_text_unifier.py
# Gera: Backend-scripts.txt
